Collecting a roofline plot without using X11 forwarding

Go to the intel web page for Intel Advisor and download the Intel Advisor for your specific OS. Note that for Mac this is reported under "Intel Advisor for Linux" where you can find the .dmg installer.

Once Intel Advisor is installed, log on Gadi

ssh user_name@gadi.nci.org.au

Then log on to a compute node 

qsub -I -P c07 -q normal -l walltime=00:10:00,mem=190GB,jobfs=100GB,ncpus=48

Once you are on a compute node, you can cd in your assignment directory, more specifically cd in the directory containing the binary of your code. Then, load specifically the following version of Intel Advisor

module load intel-advisor

Then set the following environment variables 

export ADVIXE_EXPERIMENTAL=int_roofline

export OMP_NUM_THREADS=<num_threads>

where num_thread=desired number of threads to be used.

Then run the following commands to collect all the necessary data 

advixe-cl --collect survey --project-dir ./adv_results --search-dir src:=../ --search-dir bin:=./ -- ./cloth_binary

advixe-cl --collect tripcounts -flop -enable-cache-simulation --project-dir ./adv_results --search-dir src:=../ --search-dir bin:=./ -- ./cloth_binary

This will store all the results in the adv_results directory.

Download the adv_results directory on your machine, for example using scp from your machine

scp -r user_name@gadi.nci.org.au:path_to_adv_results_directory_on_gadi destination_path_on_your_machine 

Start Intel Advisor up from your machine. Under file -> open select the "Results..." option and open the .advixeproj file. Then select the "Show my results" option. You will be able to visualise (or dump as .html) your roofline plot under "Survey & Roofline" after clicking on the ROOFLINE option appearing on the grey bar on the left.
