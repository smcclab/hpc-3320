Collecting a roofline plot using X11 forwarding

You will need to log on Gadi with X forwarding enabled by running the line

ssh -X user_name@gadi.nci.org.au

This will take you to a login node.

In order to access an entire compute node (with a 2-socket Intel Cascade Lake architecture) for 15 minutes with X forwarding run

qsub -I -X -P c07 -q normal -l walltime=00:15:00,mem=190GB,jobfs=100GB,ncpus=48

PLEASE BE SENSIBLE, do no use too much time (that is not too many of these sessions) otherwise your colleagues might not be left with enough computer time to run calculations necessary for their assignment.

Once you are on a compute node, you can cd in your assignment directory, more specifically cd in the directory containing the binary of your code. Then, load Intel Advisor

module load intel-advisor

Set the number of threads you would like to use for your roofline analysis

export OMP_NUM_THREADS=<number of threads>

Collect an initial Intel Advisor survey

advixe-cl --collect survey --project-dir ./adv_results --search-dir src:=../ --search-dir bin:=./ -- ./cloth

Then run the Intel Advisor GUI

advixe-gui &

and follow the instructions in the Video Recording of Lecture 10 (Week 12) – Part II for producing a roofline plot.
